### **FLOOR PLAN GRAPHIC LEGEND**

**1. Tags & Identifiers**
*   **Room Tag:**
    *   *Symbol:* Rectangle with a horizontal line dividing text.
    *   *Top Text:* ROOM NAME
    *   *Bottom Text:* ROOM NUMBER (e.g., 000)
*   **Wall Tag:**
    *   *Symbol:* Hexagon/Diamond shape pointing to a wall.
    *   *Top Text:* WALL TYPE (e.g., P1x)
    *   *Bottom Text:* VARIANT (IF APPLICABLE)
*   **Door Tag:**
    *   *Symbol:* Oval/Capsule shape.
    *   *Text:* DOOR NUMBER (e.g., 000)
*   **Section Tag:**
    *   *Symbol:* Circle with a filled arrow pointing in the direction of the section view.
    *   *Top Text:* VIEW NUMBER (e.g., 1)
    *   *Bottom Text:* SHEET NUMBER (e.g., A00-00)
    *   *Annotation:* "SIM" indicates SIMILAR (IF APPLICABLE)
*   **Exterior Elevation Tag:**
    *   *Symbol:* Circle with a filled triangle arrow attached.
    *   *Top Text:* VIEW NUMBER
    *   *Bottom Text:* SHEET NUMBER
    *   *Annotation:* "SIM" indicates SIMILAR (IF APPLICABLE)
*   **Interior Elevation Tag:**
    *   *Symbol:* Circle with an arrow pointing towards a wall.
    *   *Inside Text:* VIEW NUMBER (TYP.)
    *   *Outside Text:* SHEET NUMBER
*   **Exterior Wall Section Tag:**
    *   *Symbol:* Circle with a filled arrow and a tail line extending back.
    *   *Top Text:* VIEW NUMBER
    *   *Bottom Text:* SHEET NUMBER
    *   *Annotation:* "SIM" indicates SIMILAR (IF APPLICABLE)
*   **Callout Tags:**
    *   *Symbol:* Rounded rectangle (bubble) connected to a circle tag.
    *   *Circle Top:* VIEW NUMBER
    *   *Circle Bottom:* SHEET NUMBER
    *   *Annotation:* "SIM" indicates SIMILAR (IF APPLICABLE)
*   **Matchline Tags:**
    *   *Symbol:* Long dashed line with text above and below.
    *   *Format:* "MATCHLINE, SEE: [View Number]/[Sheet Number]"
    *   *Top Components:* VIEW NUMBER (ABOVE MATCHLINE) / SHEET NUMBER (ABOVE MATCHLINE)
    *   *Bottom Components:* SHEET NUMBER (BELOW MATCHLINE) / VIEW NUMBER (BELOW MATCHLINE)
